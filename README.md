# macOS Admin Scripts

A small collection of shell scripts for macOS administration and device setup.

These scripts are intended for use with management tools such as MDM and RMM platforms.

## Script Categories

### Device Management & Deployment
Install and configure management and login components.

Examples:

- Install_NinjaRMM.sh
- JamfConnect_Login_Installer_Latest.sh

## Notes
Scripts are provided as-is. Review before deploying in production.
